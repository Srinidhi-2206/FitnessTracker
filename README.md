# Fitness Tracker

A simple C++ based Fitness Tracker application developed to manage and store daily fitness activities using file handling concepts. This project helps users keep track of workouts, exercise details, and fitness records through a menu-driven interface.

## Features

- Add daily fitness records
- View saved workout details
- Store data using file handling
- Simple and user-friendly menu system
- Easy to manage and update records

## Technologies Used

- C++
- File Handling
- Object-Oriented Programming (OOP)

## How to Run the Project

1. Clone the repository
2. Open the project in VS Code or any C++ IDE
3. Compile the source file using a C++ compiler
4. Run the executable file

## Project Objective

The objective of this project is to understand the implementation of file handling and basic data management in C++. It also improves problem-solving skills and knowledge of menu-driven applications.

## What I Learned

- Working with file handling in C++
- Managing user input and records
- Building menu-driven applications
- Improving logical thinking and coding structure
