About

What is this project?

The Fitness Tracker is a menu-driven C++ program designed to help users maintain a record of their daily fitness data.

It uses structures to organize data and file handling (fstream) to permanently store and retrieve fitness records. Users can also search for specific entries using a date.

"Demonstrates how real-world applications are built using programming fundamentals — structures, functions, loops, and file I/O."

🎯

Project Objective

To develop a simple console-based application that allows users to record and monitor their daily fitness activities such as steps taken, calories burned, and exercise duration — with data persistence using file handling.

Features

What can it do?

Five core capabilities that make this tracker functional and user-friendly.

➕

Add Daily Data

Log your steps, calories burned, and workout duration for any date with a simple input prompt.

📋

View All Records

Display a complete history of all stored fitness entries retrieved from the data file.

🔍

Search by Date

Quickly find and display a specific fitness record by entering its date.

💾

Persistent Storage

All data is saved to a file using fstream so records are never lost between sessions.

🖥️

Menu-Driven UI

Simple, numbered console menu makes the app easy to navigate without any prior technical knowledge.

Working

How does it work?

A step-by-step walkthrough of the program's execution flow.

1

Launch the Application

The user opens the console application using a C++ compiler like CodeBlocks, VS Code, or Turbo C++.

2

Menu is Displayed

A numbered menu appears with options: Add Data, View Records, Search by Date, and Exit.

3

Enter Fitness Details

The user inputs today's date, steps taken, calories burned, and exercise duration in minutes.

4

Data Saved to File

The program writes the record to a .dat or .txt file using fstream, ensuring permanent storage.

5

Retrieve & Search

The user can view all records or search for a specific entry by date at any future session.
